# express-ts-boilerplate

launch project -> DEV
`yarn dev`

launch project -> DEPLOY
`yarn start`

launch project -> TEST
`yarn test`

### Tech Stack:

1. Node.js
2. Express
3. Mongoose --> database (MongoDB)
4. basic security (best preactices)
5. Eslint / Prettier
6. Redis --> cache
7. Global error handling
8. Prisma ORM --> database (PostgreSQL)
